# AWS Static Website Hosting with S3 and CloudFront

This project demonstrates how to host a static website on AWS S3 and serve it via CloudFront.

To get it up and running:

```sh
# Navigate to the terraform directory
cd 01-devops-project/terraform
# Initialize Terraform (downloads the AWS provider)
terraform init
# Review the execution plan (what Terraform will create)
terraform plan
# Apply the changes to your AWS account (type 'yes' when prompted)
terraform apply
```
